INSTALLATION (2 EASY STEPS):

1. Extract this zip file

2. Double-click "INSTALL.bat"

3. Wait 2 minutes - Done!

═══════════════════════════════════════════════════════════

That's it! The installer will:
  ✓ Download and install Python (if needed)
  ✓ Install Cite-Agent
  ✓ Create desktop shortcut
  ✓ Add to system PATH

═══════════════════════════════════════════════════════════

AFTER INSTALLATION:

Use Cite-Agent by:
  • Double-clicking desktop icon "Cite-Agent"
  • Typing "cite-agent" in any terminal
  • R Studio: Terminal pane → type "cite-agent"

═══════════════════════════════════════════════════════════

TROUBLESHOOTING:

If you see "cannot be loaded because running scripts is disabled":

  1. Right-click "Cite-Agent-Installer.ps1"
  2. Select "Properties"
  3. Check the box "Unblock" at the bottom
  4. Click "OK"
  5. Try again (Right-click → Run with PowerShell)

═══════════════════════════════════════════════════════════

Still not working? Use the web installer instead:

  1. Open PowerShell
  2. Copy and paste this command:

     iwr -useb https://raw.githubusercontent.com/Spectating101/nocturnal-archive/production-backend-only/install-clean.ps1 | iex

  3. Press Enter
  4. Wait 2 minutes
